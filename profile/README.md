## 📋 Overview

<a href="#acDSeePhotoStudio">ACDSee Photo Studio</a> combines comprehensive digital asset management with professional photo editing in a single application — serving photographers who need both a capable library management system and nondestructive RAW editing without subscribing to Adobe's Creative Cloud.

The <a href="#acDSeePhotoStudio">digital asset management system</a> organizes large photo libraries through a catalog approach — importing photos while preserving their original folder structure, adding metadata, keywords, star ratings, color labels, and categories that make large collections searchable and navigable. The catalog indexes file metadata and custom tags without moving or copying files, working with photos in their existing folder structure. The <a href="#acDSeePhotoStudio">nondestructive RAW editing</a> processes RAW files from major camera manufacturers with a full adjustment toolset — exposure, white balance, tone curves, highlight and shadow recovery, noise reduction, sharpening, lens correction, and color grading — with all edits stored as instructions that apply at export rather than modifying the original RAW file.

The <a href="#acDSeePhotoStudio">face detection and recognition</a> identifies people in photos across the library — assigning names to detected faces and finding all photos containing specific people without manual tagging. This automatic face-based organization creates a person-centric library navigation layer alongside the folder and metadata organization. The <a href="#acDSeePhotoStudio">geolocation and map view</a> displays photos on a world map based on GPS metadata embedded in files, and enables adding location data to photos taken without GPS — building a geographic organization layer across the library for location-based browsing and search.</p>

<a href="#acDSeePhotoStudio">Layer-based editing</a> in ACDSee's Edit mode creates complex composites and retouching using multiple layers with blend modes, opacity control, and layer masks — enabling Photoshop-style compositing workflows within ACDSee without launching a separate application. <a href="#acDSeePhotoStudio">Frequency separation retouching</a> provides the professional portrait retouching technique that separates texture from color and tone — enabling smooth skin retouching that preserves natural skin texture rather than the plastic appearance of simpler smoothing tools.

<a href="#acDSeePhotoStudio">Batch processing automation</a> applies adjustments, conversions, watermarks, and renaming operations to large photo sets simultaneously — processing entire shoot batches with consistent settings or preparing images for specific delivery requirements without per-image manual work. <a href="#acDSeePhotoStudio">Smart collections and filtering</a> create dynamic virtual albums based on metadata criteria — automatically collecting five-star photos, images from specific dates, photos with specific keywords, or any metadata-based filter into organized collections that update automatically as the library grows.</p>
